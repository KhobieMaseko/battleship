## Features
- Player vs Computer gameplay
- Interactive game board with visual feedback
- Responsive design for all devices
- Custom naval battle theme

## Technologies
- JavaScript (ES6+)
- Webpack
- Sass
- Babel
- Jest (for testing)

## Installation
```bash
git clone https://github.com/khobiemaseko/battleship.git
cd battleship
npm install
npm start
```
